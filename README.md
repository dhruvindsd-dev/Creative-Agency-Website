## Run Locally

Clone the project

```bash
  git clone https://github.com/dhruvindsd-dev/Creative-Agency-Website
```

Go to the project directory

```bash
  cd Creative-Agency-Website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
